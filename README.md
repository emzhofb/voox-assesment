# voox-assesment

start projects

1. clone the git
```
git@github.com:emzhofb/voox-assesment.git
```
2. move to directory
```
cd voox-assesment
```
3. install server and run (type one by one)
```
cd server
npm install
npm start
```
3. install client and run (type one by one)
```
cd ../
cd client
npm install
npm start
```

the server will running on port 3001 and the client on port 3000
